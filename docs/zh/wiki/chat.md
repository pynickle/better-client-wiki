# 聊天

### **1.** *聊天栏升高*  (1.21.1+)
* 当你拥有盔甲值时，聊天栏会向上移动十个像素以避免和盔甲值显示重叠。

### **2.** *更长的聊天历史*  (1.21.1+)
* 将聊天历史记录从原版的 100 条消息扩展至 4096 条。

### **3.** *聊天记录保留*  (1.21.1+)
* 离开世界将不再清除聊天历史。

### **4.** *持久化聊天记录*  (26.1+)
* 按世界 / 服务器分别保存聊天记录，并可在重进世界或重启游戏后恢复。

### **5.** *已恢复历史分隔符*  (26.1+)
* 可在恢复的聊天记录与新收到的消息之间插入可自定义的分隔符，并支持占位符与颜色格式化。

### **6.** *@提及补全*  (26.1+)
* 在聊天输入框中输入 `@` 时，会复用原版命令补全面板来补全当前服务器在线玩家名称。
* 选中补全后会插入 `@name `（末尾自动补一个空格），并将 mention 片段按可配置颜色高亮。
* 若整条输入以 `/` 开头，则不会触发该功能；同时支持在配置中整体开关此功能。

### **7.** *时间戳显示*  (1.21.1+)
* 在每条聊天消息前显示时间。

### **8.** *聊天格式化*  (1.21.1, 1.21.9+)
* 自动将 {pos} 等占位符替换为玩家实际坐标
