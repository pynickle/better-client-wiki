# 屏幕

### **1.** *更好的延迟显示*  (1.21.1+)
* 以数字显示玩家延迟。

### **2.** *生物群系标题*  (1.21.1+)
* 进入新生物群系时显示标题。
* 可以设置冷却时间，这样在一定冷却时间内进入新生物群系将不会显示标题。
* 可以配置标题的位置、颜色和大小。
* 可以选择显示模组名称，与其他模组兼容。

### **3.** *书籍滚动*  (1.21.1, 1.21.6+)
* 使用鼠标滚轮翻动书页。

### **4.** *不显示实验性警告*  (1.21.6+)
* 不显示创建/进入实验性世界的警告。

### **5.** *书籍保存确认*  (1.21.1, 1.21.6+)
* 关闭修改过的书籍时提示保存。

### **6.** *世界游玩时间*  (1.21.9+)
* 在世界选择界面显示每个世界的游玩时长。

### **7.** *更低的盾牌*  (1.21.9+)
* 盾牌位置更低，不会阻挡视野。

### **8.** *图腾条*  (1.21.9+)
* 显示一个条形图标，用于指示你当前拥有的图腾数量。
